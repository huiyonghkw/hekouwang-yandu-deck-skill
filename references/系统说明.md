# 演读 DECK · 沉浸式演示发布系统

把「演示版」HTML（一屏一镜、可翻页、能自动播放的 keynote 长文）打包发布到 Cloudflare Pages。
**线上：https://hekouwang.pages.dev** ｜ 出品：会勇禾口王的AI笔记。

> 这是一个**自包含**的发布系统目录，后续会升级成 Skill。演示版「内容」（各 EP 的 HTML）放在项目内的选题目录里（`算力账本/`、`偷师AI大佬/`…），本目录只放**发布引擎 + 首页 + 字体管线**。

## 目录结构

```
演读DECK/
├── publish.py        # 发布脚本：构建 dist/ → 部署 CF Pages（零三方依赖，仅标准库）
├── home.html         # 首页源（改首页改这个；dist/ 每次构建会清空，别在里面改）
├── README.md         # 本说明
├── fonts/
│   ├── src/          # 字体源：思源黑 NotoSansSC-{400,500,700,900}.woff2 + 宋 -700 + Anthropic Sans/Mono
│   └── cache/        # 子集化产物缓存（无字体源/无 venv 时回退用）
├── tools/fenv/       # 子集化用的 Python venv（fonttools+brotli）；移动会失效，丢了就重建
└── dist/             # ← 构建产物 = 部署到 CF Pages 的目录（可整删，重新构建即生成）
```

## 用法

```bash
# 从项目根（Pi.dev/）运行
python3 演读DECK/publish.py              # 构建 + 部署到 Cloudflare Pages（hekouwang）
python3 演读DECK/publish.py --build-only # 只生成 dist/，不部署

# 本地离线预览（dist 用站内 /fonts/ 绝对路径，必须起服务器，不能 file://）
cd 演读DECK/dist && python3 -m http.server 8000   # 浏览器开 http://localhost:8000/
```

加一期演示：在 `publish.py` 的 `MANIFEST` 加一行（源 HTML 路径 + slug + 系列 + 标题…），重跑即可。
加一个系列：再补 `SERIES_META` 一条。

## 它做了什么（关键设计）

1. **干净 URL**：演示版拷进 `dist/<slug>.html`（如 `suanli/ep01`），CF Pages 自动去 `.html`。
2. **字体全自托管、零 CDN**（`localize()`）：删掉源 HTML 里所有 Google/loli 外链，注入指向 `/fonts/` 的本地 `@font-face`；Anthropic 字体绝对路径也改写成 `/fonts/`。
3. **思源黑/宋按用字子集化**（`subset_cjk()`）：用 `tools/fenv` 的 `pyftsubset` 把全站实际用到的字切成小 woff2（各 ~180KB），思源黑 4 字重（400/500/700/900，800 映射 900）+ 宋（金句衬线）。无 venv/字体源时回退 `fonts/cache/`。
4. **首屏字体预加载 + 强缓存**：每页 `<head>` 注入 4 条 `<link rel=preload>`（900/400/Anthropic），消除「先系统字、后换思源」的字重跳变；`dist/_headers` 给 `/fonts/*` 设一周强缓存。

## 重建 venv（丢了或换机器时）

```bash
cd 演读DECK && python3 -m venv tools/fenv && tools/fenv/bin/pip install fonttools brotli
```

## 字体源出处

思源黑/宋来自 `@fontsource@5.0.0`（jsdelivr）逐字重静态 woff2；Anthropic Sans/Mono 来自
`~/.claude/skills/hekouwang-content-factory/assets/fonts/`（自用/演示授权，见该 Skill #9.1.3）。
