# 演读 DECK · 沉浸式演示发布系统

> **这份是「落地 README 模板」**：新项目落地时连同 `publish.py` / `home.html` 一起拷进 `<项目根>/演读DECK/README.md`，给**人**看。
> 给**模型**看的产线规矩在 SKILL.md（做 deck、换肤、留言板都从那儿走），别在这里堆细节。

把「演示版」HTML（一屏一镜、可翻页、能自动播放的 keynote 长文）打包发布到 Cloudflare Pages。
**线上：https://hekouwang.pages.dev** ｜ 出品：会勇禾口王的AI笔记。

演示版「内容」（各 EP 的 HTML）放在项目内的选题目录里（`EP-算力账本/`、`EP-偷师AI大佬/`…），本目录只放**发布引擎 + 首页 + 字体管线 + 留言板后端**。

## 目录结构

```
演读DECK/
├── publish.py       # 发布脚本：构建 dist/ → 部署 CF Pages（零三方依赖，仅标准库）
├── home.html        # 首页源（改首页改这个；dist/ 每次构建会清空，别在里面改）
├── wrangler.toml    # CF Pages 配置 + D1 绑定（留言板用）
├── schema.sql       # 留言板 D1 表结构
├── functions/
│   └── api/comments.js   # 留言板后端（Pages Functions）
├── README.md        # 本说明
├── fonts/
│   ├── src/         # 字体源：思源黑 NotoSansSC-{400,500,700,900}.woff2 + 宋 -700
│   │                #        + Anthropic Sans/Mono + Mozilla Headline/Text VF
│   └── cache/       # 子集化产物缓存（无字体源/无 venv 时回退用）
├── tools/fenv/      # 子集化用的 Python venv（fonttools+brotli）；移动会失效，丢了就重建
└── dist/            # ← 构建产物 = 部署到 CF Pages 的目录（可整删，重新构建即生成）
```

## 用法

```bash
# 从项目根运行
python3 演读DECK/publish.py              # 构建 + 部署到 Cloudflare Pages
python3 演读DECK/publish.py --build-only # 只生成 dist/，不部署

# 本地离线预览（dist 用站内 /fonts/ 绝对路径，必须起服务器，不能 file://）
cd 演读DECK/dist && python3 -m http.server 8000   # 浏览器开 http://localhost:8000/
```

**加一期**：在 `publish.py` 的 `MANIFEST` 加一行（源 HTML 路径 + slug + 系列 + 标题…），重跑即可。
**加一个系列**：补 `SERIES_META` 一条，**并手改 `home.html` 的系列卡**（首页卡是静态手写的，不由 MANIFEST 生成）。

## 它做了什么（关键设计）

1. **干净 URL**：演示版拷进 `dist/<slug>.html`（如 `suanli/ep01`），CF Pages 自动去 `.html`。
2. **字体全自托管、零 CDN**（`localize()`）：删掉源 HTML 里所有 Google/loli 外链，注入指向 `/fonts/` 的本地 `@font-face`；`FONT_ABS_RE` 把任意绝对字体路径 / 未替换的 `{{SKILL_DIR}}` 占位符统一收编成 `/fonts/`（覆盖 Anthropic + Mozilla）。
3. **思源黑/宋按用字子集化**（`subset_cjk()`）：用 `tools/fenv` 的 `pyftsubset` 把全站实际用到的字切成小 woff2（各 ~180KB），思源黑 4 字重（400/500/700/900，800 映射 900）+ 宋（金句衬线）。Mozilla VF 是拉丁、体积小，整包打进不子集化。无 venv/字体源时回退 `fonts/cache/`。
4. **首屏字体预加载 + 强缓存**：每页 `<head>` 注入 `<link rel=preload>`（含 Mozilla 的走 `PRELOAD_LINKS_V6`，按页面是否含 "Mozilla" 判定），消除「先系统字、后换思源」的字重跳变；`dist/_headers` 给 `/fonts/*` 设一周强缓存。
5. **留言板注入**（`inject_comments()`）：构建层把右侧滑出面板注入每篇 `</body>` 前，**EP 源文件不动**。

## 首次落地 / 换机器

```bash
# 子集化 venv（丢了或换机器时重建）
cd 演读DECK && python3 -m venv tools/fenv && tools/fenv/bin/pip install fonttools brotli

# 留言板 D1（不做留言板可跳过，并删掉 wrangler.toml / schema.sql / functions/）
wrangler d1 create <你的库名>                                # 拿 database_id 填进 wrangler.toml
wrangler d1 execute <你的库名> --remote --file=schema.sql    # 建表

# 部署前先登录
wrangler login
```

**要填的自己的值**：`publish.py` 的 `CF_BEACON_TOKEN`（CF Web Analytics，留空则不注入）、`wrangler.toml` 的 `name` / `database_name` / `database_id`。
